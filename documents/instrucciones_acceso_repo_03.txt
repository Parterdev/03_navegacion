DESCRIPCIÓN DEL PROYECTO

Aplicación móvil desarrollada con React Native CLI, TypeScript y React Navigation para implementar una navegación multipantalla mediante navegación por pila y navegación por pestañas inferiores.

La aplicación permite visualizar un catálogo básico de productos tecnológicos. Desde la pantalla principal, el usuario puede seleccionar un producto y navegar hacia una pantalla de detalle, donde se muestra la información completa del item seleccionado.

El proyecto integra Bottom Tabs para separar las secciones principales de la aplicación y Stack Navigator dentro de la pestaña Catálogo para navegar entre la lista de productos y el detalle. Además, se implementa el paso de datos entre pantallas mediante parámetros de ruta.

El proyecto fue desarrollado tomando como referencia el documento de la actividad ubicado en la carpeta documents/ del repositorio.

INSTRUCCIONES DE INSTALACIÓN Y EJECUCIÓN

1. Clonar el repositorio:

git clone https://github.com/Parterdev/03_clase_tres_navegacion.git

2. Ingresar al proyecto:

cd 03_clase_tres_navegacion

3. Instalar dependencias:

npm install

4. Ejecutar Metro:

npm run start

5. Ejecutar en Android:

npm run android

6. Ejecutar en iOS:

cd ios
pod install
cd ..
npm run ios

NOTA TÉCNICA

Durante las pruebas puede mostrarse el siguiente warning en React Native DevTools:

InteractionManager has been deprecated and will be removed in a future release.

Este aviso no proviene del código desarrollado en src, ya que el proyecto no utiliza InteractionManager directamente. La aplicación funciona correctamente y el warning no afecta la navegación, el paso de parámetros ni la visualización de las pantallas.
